## 1. Что такое проект

**Transkrib SmartCut AI** — десктопное приложение для Windows (Electron + FastAPI), которое:
- принимает длинное видео (файл или YouTube URL)
- транскрибирует через Whisper
- выбирает ключевые моменты через Claude API
- рендерит MP4-резюме с субтитрами через FFmpeg

Целевая аудитория: люди, которые работают с вебинарами, интервью, подкастами.

## 4. Ключевые архитектурные решения

### Pipeline обработки видео (актуальный)
```
Видео/URL
    ↓ Audio extraction (ffmpeg — сначала аудио для ускорения)
    ↓ Whisper → сегменты с timestamps
    ↓ pause_detector → фразы, разбитые по паузам > 0.8 сек
    ↓ highlight_scorer → оценка каждой фразы 1-10 (Claude API)
    ↓ Фильтр score ≥ 6 + группировка в клипы 10-30 сек
    ↓ Приветствие/прощание — всегда включены (первые/последние 30 сек)
    ↓ Параллельный FFmpeg → нарезка клипов
    ↓ Результат: MP4 + _segments.json (кэш)
```

### Transcript export (Фаза 4.6)
```
ResultGallery → кнопка [📄 Транскрипт ▾]
    ├── 👁 Смотреть в приложении → TranscriptViewer.tsx
    ├── ⬇ Скачать .TXT  (чистый текст)
    ├── ⬇ Скачать .SRT  (субтитры с таймкодами)
    └── ⬇ Скачать .JSON (segments + scores)

TranscriptViewer:
    - Фильтр: Все фразы / Только важные ≥6
    - Поиск по тексту в реальном времени
    - Цветовые полосы: зелёная (8-10) / жёлтая (5-7) / серая (1-4)
    - Кнопки скачать внизу
    - IPC: app:saveTranscript → диалог сохранения файла
```

### Preview generation (Фаза 3.4)
```
После обработки → POST /preview/generate → preview_service.py
    → Stability AI API (первые 200 символов транскрипта как prompt)
    → сохранение {task_id}_preview.jpg
    → 3 состояния в карточке: AI-превью / spinner / заглушка
    → кнопка ↻ refresh при hover
```

### Export (Фаза 3.3)
```
ResultGallery → [⬆ Экспорт] → ExportModal.tsx
    Настройки: формат (MP4/MKV/WebM), качество (CRF 18/23/28),
               разрешение (Оригинал/1080p/720p/480p),
               субтитры (встроить/SRT/оба/нет),
               папка сохранения
    → POST /export → FFmpeg
```

## 6. Известные ограничения и gotchas

### OneDrive EEXIST
Файлы проекта на OneDrive. Инструмент Edit иногда падает с EEXIST.
**Workaround**: писать в `C:/Users/Admin/AppData/Local/Temp/`, затем `shutil.copy2()`.

### Supabase в Electron
CSP в `index.html` разрешает `https://*.supabase.co` и `wss://*.supabase.co` ✅

### TypeScript TranslationKeys
Использовать `DeepString<T>` utility type в `ru.ts`.

### PyInstaller + Windows 11
- Антивирус может блокировать backend.exe
- HKLM запись требует прав администратора при первом запуске
