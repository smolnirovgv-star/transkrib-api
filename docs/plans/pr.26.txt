Функция format_transcript(segments, language, title):
  - Отправить транскрипт в Claude API с промптом:
  
  """
  Ты редактор текста. Обработай транскрипцию речи:
  
  1. Убери слова-паразиты (ну, вот, значит, э-э, да, понимаете)
  2. Расставь знаки препинания по правилам {language}
  3. Раздели на абзацы по смыслу (каждая новая мысль)
  4. Исправь орфографию
  5. Сохрани стиль и слова спикера — не переписывай смысл
  6. Верни только обработанный текст без пояснений
  
  Транскрипция:
  {text}
  """
  
  Возвращает: отформатированный текст

2. Обнови форматы экспорта в transcript.py:

TXT формат:
  ═══════════════════════════════════════
  {title}
  Дата: {date} | Длительность: {duration}
  Транскрипт подготовлен: Transkrib SmartCut AI
  ═══════════════════════════════════════
  
  {отформатированный текст по абзацам}
  
  ───────────────────────────────────────
  © Transkrib SmartCut AI | transkrib.ai
  ═══════════════════════════════════════

SRT формат — без изменений (таймкоды важны)

JSON формат:
  {
    "title": "...",
    "date": "...",
    "language": "ru",
    "formatted_text": "...",  ← новое поле
    "segments": [...]
  }

HTML формат (НОВЫЙ):
  Красивая страница в стиле приложения:
  - Шрифт: system-ui, sans-serif
  - Цвета: #0F172A фон (dark), #7C3AED акцент
  - Заголовок с метаданными
  - Текст разбит на абзацы с таймкодами слева
  - Кнопка "Печать" (window.print())
  - Адаптивный дизайн
  
  Пример строки:
  [00:15] Сегодня разберём важный вопрос...
  [02:30] Первый ключевой принцип — это...
