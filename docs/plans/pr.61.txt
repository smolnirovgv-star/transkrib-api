КНОПКА 1 — Скачать (мгновенное скачивание):
Должна скачивать MP4 файл сразу в папку Загрузки
БЕЗ диалога выбора папки.

В браузерном режиме:
const handleDownload = () => {
  const url = 'http://127.0.0.1:8000/api/tasks/result/' 
    + encodeURIComponent(selected.filename)
  const a = document.createElement('a')
  a.href = url
  a.download = selected.filename
  document.body.appendChild(a)
  a.click()
  document.body.removeChild(a)
}

В Electron режиме:
if (window.electronAPI && window.electronAPI.downloadResult) {
  window.electronAPI.downloadResult(selected.filename)
}
