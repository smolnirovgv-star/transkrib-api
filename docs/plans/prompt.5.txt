### 5.5 — Timeline и SegmentPreview
═══════════════════════════════════════════

Создай компонент: SegmentTimeline.tsx
Путь: platforms/desktop_windows/src/renderer/components/SegmentTimeline.tsx

Показывается в ResultGallery под карточкой результата.

Визуальная шкала времени:
┌────────────────────────────────────────┐
│ 0:00          30:00          1:00:00   │
│ ████░░░░████████░░░░░████░░░████░░░░░ │
│  ^включён  ^пропущен  ^включён         │
└────────────────────────────────────────┘

- Зелёные блоки = включённые сегменты (score ≥ 6)
- Серые блоки = вырезанные паузы/неважные части
- При наведении на блок — tooltip: "00:15 - 00:43 | score: 8 | Ключевой вывод"
- Клик по сегменту → SegmentPreview модал

Создай компонент: SegmentPreview.tsx
Путь: platforms/desktop_windows/src/renderer/components/SegmentPreview.tsx

Модал с предпросмотром конкретного сегмента:
┌──────────────────────────────────────┐
│  Сегмент 3 из 12    [×]              │
│  ┌──────────────────────────────┐    │
│  │   HTML5 video player         │    │
│  │   [старт: 00:15 - 00:43]     │    │
│  └──────────────────────────────┘    │
│  Текст: "Первый ключевой принцип..." │
│  Оценка: ████████░░ 8/10             │
│  [← Пред]  [Включить/Исключить]  [След →] │
└──────────────────────────────────────┘

- Кнопка "Включить/Исключить" — помечает сегмент
- Пересчитывает итоговое видео с учётом изменений
