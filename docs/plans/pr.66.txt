ЗАДАЧА — Убрать overflow:hidden у screens-wrapper

Найди в globals.css классы:
.screens-wrapper и .screens-track

Добавь или замени:
.screens-wrapper {
  overflow: visible !important;
  height: auto !important;
}

.screens-track {
  overflow: visible !important;
}

Также убери ограничение высоты на секции How It Works:
.how-section {
  overflow: visible !important;
  height: auto !important;
  min-height: unset !important;
}
