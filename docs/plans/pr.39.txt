Контекст: Transkrib SmartCut AI
ВАЖНО: Изменять ТОЛЬКО указанные файлы. Всё остальное не трогать.

ЗАДАЧА — Сохранять задачи в файл (персистентность)

Файл: backend/app/workers/standalone_tasks.py 
(или где хранится tasks dict)

ПРОБЛЕМА: При перезапуске backend все задачи теряются.
Tasks хранятся только в памяти (dict).

ИСПРАВЛЕНИЕ:
1. При создании задачи — сохранять в JSON файл:
   %APPDATA%\Transkrib\storage\tasks.json

2. При старте backend — загружать задачи из файла

3. При обновлении задачи — обновлять файл

Формат tasks.json:
{
  "task_id_1": { все поля задачи },
  "task_id_2": { все поля задачи }
}

Не использовать backticks. Только одинарные кавычки.
Отчёт только по изменённым строкам.
