## Тарифы (test-ветка, не в prod пока)

Минутные тарифы:
- **Пробный** 0 ₽ / 7 дней / 30 мин
- **Базовый** 450 ₽ / 30 дней / 300 мин (5 ч)
- **Стандарт** 1700 ₽ / 30 дней / 1500 мин (25 ч)
- **Про** 8900 ₽ / 365 дней / 24000 мин (400 ч)

Overflow при превышении:
- Быстрый час 150 ₽ / 60 мин / на 24 ч
- 3 часа 350 ₽ / 180 мин / на 7 дней
- 10 часов 900 ₽ / 600 мин / на 30 дней

UX: warning при 90% использования, меню overflow/апгрейд при исчерпании.

## Видео-формат для обработки

Архитектурное решение: при скачивании через yt-dlp ВСЕГДА конвертировать в MP4 (H.264 + AAC) как единый формат для:
- ffmpeg-нарезки (исключает проблемы кодеков)
- выдачи пользователю (максимальная совместимость)
