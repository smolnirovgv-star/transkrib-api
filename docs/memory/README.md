# Project Memory — Transkrib

Это локальная база знаний проекта Transkrib SmartCut AI. Используется:
- Claude Code (Cursor) — читает релевантный файл по запросу пользователя или в начале сессии
- Внешним Claude через копирование/чтение через GitHub
- Самим Геннадием — чтобы быстро вспомнить контекст после паузы

## Файлы

| Файл | Когда читать |
|---|---|
| 00-architecture.md | Любой вопрос про "как устроен проект", "какие репо", "где что развёрнуто" |
| 01-keys-and-urls.md | Нужны API-ключи, URL'ы, ID сервисов, инвайт-коды |
| 02-railway-infra.md | Деплой, переменные окружения, проблемы с билдами |
| 03-cursor-rules.md | Стандартные правила для промтов Cursor — обязательно читать перед началом любой задачи |
| 04-known-bugs.md | Перед фиксом — проверить, не известный ли баг и что про него уже знаем |
| 05-roadmap.md | Планы v1.1.0+, тарифы, глобальная экспансия |
| 06-bot-flow.md | Как устроен флоу бота, какие форматы, кнопки, тарифы |
| 07-decisions-log.md | Журнал архитектурных решений с датами и обоснованиями |

## Правила обновления

- Любое изменение в проде, новый ключ, новое решение → обновить соответствующий файл в том же PR/коммите
- Удалять устаревшее, не накапливать
- Не дублировать между файлами — ссылаться: "см. 01-keys-and-urls.md"
- Дата последнего обновления — в начале каждого файла
