## 1. Общие положения

Настоящая Политика конфиденциальности описывает, как приложение **Transkrib SmartCut AI** (далее — «Приложение», «мы») собирает, использует и защищает данные пользователей (далее — «вы», «пользователь»).

Используя Приложение, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с какими-либо положениями — пожалуйста, прекратите использование Приложения.

Приложение разработано для операционной системы Windows и предназначено для автоматической обработки видеозаписей с использованием технологий искусственного интеллекта.

## 2. Какие данные мы собираем

### 2.1 Данные аккаунта (при регистрации)
- Адрес электронной почты
- Хэш пароля (никогда не хранится в открытом виде)
- Дата регистрации и последнего входа

### 2.2 Лицензионные данные
- Лицензионный ключ активации (формат TRSK-*)
- Статус лицензии (тип плана, дата активации, срок действия)
- Hardware fingerprint устройства (для привязки лицензии к машине)

### 2.3 Данные trial-периода
- Дата начала пробного периода
- Количество обработанных видео
- Статус trial (хранится локально в реестре Windows)

### 2.4 Данные обработки (только локально)
- Загружаемые видеофайлы — **обрабатываются исключительно на вашем компьютере**
- Транскрипции речи — **генерируются локально через Whisper AI**
- Результирующие видеофайлы — **сохраняются только на вашем компьютере**
- Файлы субтитров и транскриптов — **хранятся только локально**

### 2.5 Технические данные
- Версия операционной системы (для диагностики)
- Журналы ошибок (только при возникновении сбоев, хранятся локально)

## 3. Как мы используем данные

| Данные | Цель использования |
|--------|-------------------|
| Email и пароль | Аутентификация и управление аккаунтом |
| Лицензионный ключ | Проверка права использования Приложения |

| Краткое описание видео | Генерация превью через Stability AI |
| Журналы ошибок | Диагностика и устранение технических проблем |

**Мы НЕ используем ваши данные для:**
- Таргетированной рекламы
- Продажи или передачи третьим лицам в коммерческих целях
- Обучения AI-моделей на ваших видеоматериалах
- Аналитики поведения пользователя без его согласия

## 4. Передача данных третьим лицам

Для обеспечения работы Приложения мы взаимодействуем со следующими сервисами:

### 4.1 Supabase (аутентификация и хранение аккаунтов)
- **Передаётся:** email, хэш пароля, данные лицензии
- **Цель:** хранение аккаунта и управление сессиями
- **Политика:** https://supabase.com/privacy

### 4.2 Anthropic Claude API (анализ текста)
- **Передаётся:** только текст транскрипции (не видеофайл)
- **Цель:** определение ключевых моментов и оценка важности фраз
- **Политика:** https://www.anthropic.com/privacy

### 4.3 Stability AI (генерация превью)
- **Передаётся:** короткий текстовый фрагмент (до 200 символов из транскрипции)
- **Цель:** автоматическая генерация изображения-превью для результата
- **Используется:** только если задан ключ API в настройках
- **Политика:** https://stability.ai/privacy-policy

**Мы НЕ передаём третьим лицам:**
- Ваши видеофайлы
- Полные транскрипции (только фрагменты для анализа)
- Персональные данные помимо указанных выше

## 5. Локальная обработка данных

Это ключевой принцип работы Transkrib SmartCut AI:

**Все видеофайлы обрабатываются локально на вашем компьютере.**

- Загруженные видеофайлы **никогда не покидают ваш компьютер**
- Транскрипция речи выполняется **локально** через встроенную модель Whisper AI
- Нарезка и монтаж видео выполняется **локально** через FFmpeg
- Готовые видеоролики **сохраняются только на вашем устройстве**
- Все промежуточные файлы хранятся в изолированной папке:
  `%APPDATA%\Transkrib\storage\`

На внешние серверы передаётся **только текст** (транскрипция или её фрагменты) для AI-анализа — и только в случае, если вы явно запустили функцию обработки.

## 6. Хранение и защита данных

### 6.1 Локальное хранение
- Видео, транскрипты, результаты: `%APPDATA%\Transkrib\storage\`
- Лицензионный ключ: `%APPDATA%\Transkrib\storage\.license\license.key`
- Данные trial: реестр Windows (HKCU и HKLM) + файл `trial.dat`
- Сессия аккаунта: localStorage браузерного движка Electron

### 6.2 Облачное хранение (Supabase)
- Данные аккаунта хранятся на серверах Supabase (регион: EU West)
- Передача данных защищена протоколом TLS 1.3
- Пароли хранятся в виде bcrypt-хэша — оригинальный пароль недоступен

### 6.3 Защита лицензии
- Hardware fingerprint используется исключительно для привязки лицензии
- Данные подписаны алгоритмом HMAC-SHA256
- Попытки обхода защиты фиксируются в журнале безопасности

## 8. Локальное хранилище и реестр Windows

Приложение использует следующие механизмы локального хранения:

| Механизм | Что хранится | Как удалить |
|----------|-------------|-------------|
| AppData папка | Видео, транскрипты, лицензия | Удалить `%APPDATA%\Transkrib\` |
| Реестр Windows (HKCU) | Данные trial-периода | `regedit` → удалить ключ Transkrib |
| localStorage (Electron) | Сессия аккаунта, настройки | Выйти из аккаунта в приложении |

Все локальные данные остаются на вашем компьютере и полностью под вашим контролем.
