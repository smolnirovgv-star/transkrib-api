# Transkrib SmartCut AI — Руководство пользователя

**Версия:** 1.0.0
**Платформа:** Windows 10 / 11 (64-bit и 32-bit)

---

## 1. Системные требования

| Параметр | Минимум | Рекомендуется |
|----------|---------|---------------|
| ОС | Windows 10 (64-bit) | Windows 11 (64-bit) |
| ОЗУ | 4 ГБ | 8 ГБ и более |
| Диск | 2 ГБ свободных | 5 ГБ и более |
| GPU | — | NVIDIA (ускоряет транскрипцию) |
| Интернет | Требуется (для AI-анализа) | Стабильный ≥10 Мбит/с |

> **Важно:** Функции AI (Whisper, Claude) требуют подключения к интернету для загрузки модели и API-запросов.

## 2. Установка

1. Запустите **Transkrib-Setup-1.0.0.exe** (универсальный) или версию под вашу архитектуру:
   - `Transkrib-Setup-1.0.0-x64.exe` — для 64-bit Windows (рекомендуется)
   - `Transkrib-Setup-1.0.0-ia32.exe` — для 32-bit Windows

2. Следуйте шагам мастера установки:
   - Выберите папку установки (по умолчанию: `%LocalAppData%\Programs\Transkrib`)
   - Установите ярлыки на рабочий стол и в меню «Пуск»

3. Нажмите **«Установить»** и дождитесь завершения (~30 сек).

4. Нажмите **«Запустить Transkrib SmartCut AI»** для первого запуска.

> Данные приложения хранятся в: `%AppData%\Transkrib\storage\`
> Удалить данные можно через «Панель управления → Программы → Transkrib».

## 3. Первый запуск — Пробный период (Trial)

При первом запуске откроется **Мастер настройки**. У вас два варианта:

### Вариант A: Начать бесплатный пробный период
- Нажмите **«Начать пробный период (7 дней)»**
- Получите **7 дней** бесплатного доступа
- Лимиты Trial: **3 видео в день**, до **30 минут** каждое

### Вариант B: Активировать лицензионный ключ
- Введите ключ формата `TRSK-BASE-XXXX-XXXX-XXXXXXXX`
- Нажмите **«Активировать»**

> **Примечание:** Пробный период и лицензия привязаны к устройству и не требуют регистрации.

## 4. Активация лицензии

Если у вас есть лицензионный ключ:

1. На экране мастера нажмите **«Использовать лицензионный ключ»**
2. Введите ключ в поле (формат: `TRSK-PLAN-XXXX-XXXX-XXXXXXXX`)
3. Нажмите **«Активировать»**

**Планы лицензий:**

| План | Срок | Описание |
|------|------|----------|
| Trial | 7 дней | 3 видео/день, до 30 мин |
| Base (`TRSK-BASE-...`) | 10 дней | Безлимит, + история, + TXT |
| Standard (`TRSK-STND-...`) | 30 дней | + SRT/JSON, AI превью, приоритет |
| Pro (`TRSK-PREM-...`) | 365 дней | Все функции, ранний доступ |

## 5. Как обработать видео

### Шаг 1: Выбор источника
На главном экране выберите способ загрузки:

**Вкладка «Файл»:**
- Перетащите видеофайл в зону загрузки
- Или нажмите **«Выбрать файл»** и выберите файл в проводнике
- Поддерживаемые форматы: MP4, MKV, AVI, MOV, WebM

**Вкладка «Ссылка»:**
- Вставьте URL видео (YouTube, VK, и другие)
- Нажмите **«Отправить»**

### Шаг 2: Настройки обработки (опционально)
Нажмите иконку ⚙️ (шестерёнка) для настройки:
- **Скорость транскрипции:** `tiny` (быстро), `base` (баланс), `small`/`medium` (точно)
- **Длина итогового видео:** 20%, 40%, 60% от оригинала

### Шаг 3: Запуск и мониторинг
После загрузки начнётся автоматическая обработка:

| Этап | Описание |
|------|----------|
| 🔄 Конвертация | Видео конвертируется в MP4 |
| 🎙️ Транскрипция | Whisper AI распознаёт речь и создаёт таймкоды |
| ⏸️ Анализ пауз | Алгоритм находит границы фраз |
| 🧠 Оценка важности | Claude AI оценивает каждую фразу (1–10) |
| ✂️ Нарезка | FFmpeg собирает лучшие фрагменты (score ≥ 6) |
| 📦 Сборка | Итоговое видео компилируется |

Прогресс отображается в реальном времени. Среднее время: **3–10 минут** на 1 час видео.

### Шаг 4: Результат
После завершения откроется экран с карточкой результата:
- **▶ Нажмите на карточку** — воспроизведение в приложении
- **⬇ Скачать** — сохранить MP4 на диск
- **📄 Транскрипт** — просмотр или скачивание текста

## 6. Экспорт результата

Нажмите кнопку **«Экспорт»** (иконка ↑) на карточке результата:

### Настройки экспорта:
- **Формат:** MP4
- **Качество:** Высокое / Среднее / Низкое
- **Разрешение:** Оригинал / 1080p / 720p
- **Субтитры:** Встроить в видео / Отдельный .srt / Оба / Без субтитров
- **Папка:** Выберите папку сохранения

Нажмите **«Экспортировать»** — файл сохранится в выбранную папку.

## 7. Транскрипт

### Просмотр в приложении
1. На карточке результата нажмите **📄** (иконка файла)
2. Выберите **«Смотреть в приложении»**
3. Откроется TranscriptViewer с:
   - Фильтром «Все фразы» / «Только важные ≥6»
   - Поиском по тексту в реальном времени
   - Цветными полосами важности (🟢 8–10, 🟡 5–7, ⬜ 1–4)
   - Таймкодами для навигации

### Карта сегментов (Timeline)
На карточке результата нажмите **«Карта сегментов»**:
- Зелёные блоки = включённые фрагменты (score ≥ 6)
- Серые блоки = вырезанные части
- **Нажмите на блок** — предпросмотр конкретного сегмента

### Скачивание транскрипта
Нажмите **📄 → выберите формат:**
- **TXT** — чистый текст, каждая фраза с новой строки
- **SRT** — субтитры с таймкодами (для видеоредакторов)
- **JSON** — полные данные с оценками (для разработчиков)

## 9. Часто задаваемые вопросы (FAQ)

**Q: Видео обрабатывается очень долго, что делать?**
A: Время зависит от длины видео и модели Whisper. Для быстрой обработки выберите модель `tiny` в настройках. Первый запуск занимает дольше — Whisper загружает модель (~460 МБ).

**Q: Ошибка «Claude вернул пустой ответ»**
A: Проверьте наличие API-ключа Anthropic в файле `backend/.env`. Убедитесь в наличии интернета.

**Q: Можно ли обрабатывать несколько видео одновременно?**
A: Нет, обработка происходит последовательно. Дождитесь завершения текущего.

**Q: Где хранятся обработанные видео?**
A: В папке `%AppData%\Transkrib\storage\results\`

**Q: Как удалить все данные приложения?**
A: Удалите папку `%AppData%\Transkrib\` вручную или через «Программы и компоненты» — установщик предложит это при удалении.

**Q: Приложение не запускается на 32-bit Windows**
A: AI-функции (Whisper/PyTorch) работают только на 64-bit. На 32-bit доступен базовый интерфейс, но обработка видео недоступна.

**Q: Нужен ли интернет постоянно?**
A: Интернет нужен для: загрузки модели Whisper (один раз), запросов к Claude API (каждое видео), загрузки видео по URL. После загрузки модели транскрипция работает офлайн.

**Q: Пробный период закончился, что делать?**
A: Приобретите лицензионный ключ на сайте или через поддержку. Введите ключ в мастере настройки.
